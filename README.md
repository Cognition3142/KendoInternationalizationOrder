# load order demo

Demo to display that the order of the internationalisation load files will effect the outcome of using locales.

The breaking code in currently active in app.module.ts and the working code is commented out.
If you switch which code block is commented, you can see that order matters.

### Try it

1. Clone the repo
2. `npm i`
3. `npm start`
4. Navigate to `http://localhost:8080`
